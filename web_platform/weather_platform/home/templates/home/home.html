Welcome to the Weather Data Project

Project Overview

This project is a comprehensive system to collect, process, and display weather data. The system is built using a combination of Django, an ETL pipeline, and various data processing techniques.

API

The system pulls real-time weather data from a weather API, which is then stored in a PostgreSQL database for further processing and analysis.

ETL Pipeline

The ETL (Extract, Transform, Load) pipeline is responsible for fetching, cleaning, and loading data into the database. This ensures that only relevant and accurate data is stored for use in the system.

Dashboard

Our weather dashboard provides an intuitive interface to visualize weather data, making it easy to track current conditions, forecasts, and other important metrics.
